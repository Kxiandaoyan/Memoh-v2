# 快速上手

本章帮助你在安装完成后，从登录到发送第一条消息，3 分钟内完成。

## 1. 登录

打开浏览器访问 `http://<你的服务器IP>:8082`，进入登录页面。

- **用户名**：在 `config.toml` 的 `[admin]` 段配置的管理员账号。
- **密码**：对应的管理员密码。

登录成功后自动跳转到 **对话** 页面。

## 2. 认识导航

左侧边栏从上到下依次为：

| 导航项 | 说明 |
|--------|------|
| 系统设置 | 语言、主题、时区、系统诊断 |
| 对话 | 与 Bot 实时对话 |
| Bots | 查看和管理所有 Bot |
| 共享工作区 | 跨 Bot 共享的文件目录 |
| 定时任务 | 查看所有 Bot 的定时任务 |
| 模型管理 | 管理 LLM 提供方和模型 |
| 搜索提供方 | 配置搜索引擎（如 Brave） |
| Token 用量 | 查看 Token 消耗统计 |
| 流程日志 | 查看每轮对话的处理步骤 |
| 管理员设置 | 个人资料、密码、渠道绑定 |

左下角显示当前登录用户的头像和名称。

## 3. 创建第一个 Bot

1. 点击左侧 **Bots** 进入 Bot 列表页面。
2. 点击右上角 **创建 Bot** 按钮。
3. **选择模板**（可选）：
   - 从分类标签中筛选模板（如 productivity、assistant、creative 等）。
   - 也可以选择 "空白 Bot"，不使用任何模板。
4. 点击 **下一步**，填写基本信息：
   - **显示名称**（必填）：Bot 的名字。
   - **头像 URL**（可选）：一个图片链接。
   - **类型**：`personal`（私人，仅自己可用）或 `public`（公开，其他用户也可访问）。
   - **特权模式**（可选）：开启后 Bot 拥有更多系统权限。
5. 点击 **创建**，等待容器初始化完成（通常几秒钟）。

## 4. 配置模型

新建的 Bot 需要指定 AI 模型才能对话：

1. 先到 **模型管理** 页面添加至少一个 LLM 提供方（如 OpenAI）和对应模型。
2. 回到 **Bots** → 点击 Bot 卡片进入详情 → 切换到 **设置** 标签。
3. 在 **模型配置** 中选择对话模型（chat model）。
4. 点击 **保存**。

## 5. 发送第一条消息

1. 点击左侧 **对话** 导航。
2. 左侧面板列出所有可用的 Bot，点击你刚创建的 Bot。
3. 在底部输入框中输入消息，按 **Enter** 发送。
4. Bot 的回复会以流式方式实时显示。
