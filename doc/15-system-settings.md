# 系统设置

## 进入系统设置

点击左侧导航最顶部的 **系统设置** 进入。系统设置影响整个平台的行为，对所有 Bot 生效。

## 显示设置

### 语言

下拉选择界面语言：
- **中文** (zh)
- **English** (en)

切换后界面立即生效，无需刷新。

### 主题

下拉选择界面主题：
- **亮色** (light)
- **暗色** (dark)

## 时区设置

下拉选择服务器时区。此时区影响：
- 定时任务的 Cron 表达式解析。
- Bot 系统提示词中注入的当前时间。
- 流程日志和其他时间显示。

可选的时区包括：
- UTC
- Asia/Shanghai（北京时间）
- Asia/Tokyo
- Asia/Singapore
- America/New_York
- America/Los_Angeles
- Europe/London
- 等 20+ 常用时区

修改后会立即通知后端更新，无需重启服务。

## 系统诊断

点击 **运行诊断** 按钮，系统会检查各个基础服务的连接状态。

### 诊断结果

诊断完成后会显示整体状态：
- **Healthy**：所有服务正常。
- **Degraded**：部分服务有问题。
- **Unhealthy**：核心服务不可用。

### 检查项

每个检查项显示：
- 服务名称
- 状态（ok / warn / error）
- 延迟（毫秒）

诊断帮助你快速排查服务器问题，例如数据库连接失败、向量数据库不可达等。
