# 子智能体

Bot 详情页 → **子智能体** 标签，管理 Bot 的子智能体（Subagents）。

## 什么是子智能体

子智能体是附属于主 Bot 的辅助 AI 实体。主 Bot 可以将特定任务委派给子智能体处理，实现任务分工和专业化。

每个子智能体拥有：
- 独立的名称和描述
- 指定的技能集

## 查看子智能体

子智能体以卡片列表展示，每张卡片包含：
- **名称**
- **描述**
- **技能列表**（以标签形式展示）
- **创建时间**

## 创建子智能体

点击 **创建** 按钮，填写：

| 字段 | 说明 |
|------|------|
| 名称 | 子智能体的标识名 |
| 描述 | 子智能体的职责和用途说明 |
| 技能 | 以逗号分隔的技能列表 |

创建后，主 Bot 在对话过程中可以根据需要自动调用对应的子智能体。

## 编辑子智能体

点击子智能体卡片，在弹出的对话框中修改名称、描述或技能。

## 删除子智能体

点击卡片上的删除按钮即可移除。
