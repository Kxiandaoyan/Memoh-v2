# 管理员设置

## 进入管理员设置

点击左侧导航底部的 **管理员设置** 进入。此页面管理当前登录用户的个人信息和安全设置。

## 用户资料

页面顶部显示当前用户的头像（支持 fallback 到名称首字母）。

### 基本信息

| 字段 | 说明 | 可编辑 |
|------|------|--------|
| 用户 ID | 系统分配的唯一标识 | 否（只读） |
| 用户名 | 登录用户名 | 否（只读） |
| 显示名称 | 在界面中展示的名字 | 是 |
| 头像 URL | 头像图片链接 | 是 |

修改后点击保存按钮提交。

## 修改密码

| 字段 | 说明 |
|------|------|
| 当前密码 | 输入当前密码进行验证 |
| 新密码 | 输入新密码 |
| 确认密码 | 再次输入新密码以确认 |

两次输入的新密码必须一致才能提交。

## 渠道身份绑定

### 已绑定的渠道

列表展示当前用户已绑定的所有平台身份，每个条目显示：
- 平台名称（如 Telegram、Feishu）
- 平台身份标识

### 生成绑定码

绑定码用于将即时通讯平台的账号与 Memoh 用户关联。

操作步骤：
1. 选择目标平台（可选择特定平台或"所有平台"）。
2. 设置有效期（默认 3600 秒 = 1 小时）。
3. 点击 **生成** 按钮。
4. 复制生成的绑定码。
5. 在对应平台（如 Telegram）中，向 Bot 发送绑定码。
6. 绑定成功后，该平台的消息会关联到你的 Memoh 用户账号。

绑定完成后，无论你从 Web、Telegram 还是飞书与 Bot 对话，系统都能识别为同一个用户，共享对话历史和记忆。

## 退出登录

页面底部的 **退出登录** 按钮，点击后清除登录状态并跳转到登录页面。
