# 定时任务

## 进入定时任务页面

点击左侧导航的 **定时任务** 进入。

## 页面结构

定时任务页面按 Bot 分组展示。每个有定时任务的 Bot 作为一个可折叠的分组（默认展开），下方列出该 Bot 的所有定时任务。

如果某个 Bot 没有定时任务，则不会出现在此页面。

## 任务信息

每个定时任务显示以下信息：

| 字段 | 说明 |
|------|------|
| 启用开关 | 控制任务是否激活 |
| 名称 | 任务名称，旁边有状态徽标 |
| 描述 | 任务的说明（如果有） |
| Cron 表达式 | 触发时间规则，旁边显示服务器时区 |
| 命令 | 任务触发时发送给 Bot 的提示词 |
| 执行次数 | 已执行次数 / 最大次数（如果设置了上限），否则显示"无限制" |

## 操作

### 启用/停用任务

每个任务行左侧有一个开关，切换即可启用或停用。

### 删除任务

点击任务行右侧的删除按钮（垃圾桶图标），确认后删除。

## 定时任务的创建

定时任务由 Bot 在对话中通过内置工具 `create_schedule` 自行创建，而非在 Web 界面手动添加。

你可以在对话中告诉 Bot：
> "每天早上 9 点提醒我查看邮件"

Bot 会调用 `create_schedule` 工具创建对应的定时任务，之后你可以在定时任务页面查看和管理。

Bot 还可以通过 `update_schedule`、`delete_schedule`、`list_schedules`、`get_schedule` 等工具管理已有任务。

## 时区说明

Cron 表达式旁边显示的时区来自 **系统设置** 中配置的服务器时区。确保时区设置正确，否则定时任务可能在非预期的时间触发。
