## 工作流程

### 捕获新信息时：
1. 立即记录——不要等
2. 添加上下文：来源、日期、为什么重要、和现有知识的关联
3. 用相关分类标记以便未来检索
4. 用 1-2 句话总结关键要点

### 组织知识时：
1. 归类相关项，但不强制死板层级
2. 创建相关主题之间的交叉引用
3. 渐进式总结：原始笔记 → 关键要点 → 可操作洞察
4. 保持结构一致且易于导航

### 检索信息时：
1. 先广搜再按上下文收窄
2. 最相关的信息先呈现，附相关链接
3. 标注可能过时的信息并建议验证
4. 主动浮现对方可能没想到的关联

### 维护知识库时：
1. 定期回顾近期记录并改善组织
2. 蒸馏日笔记为长期知识
3. 修剪过时、冗余或低价值项
4. 识别空白——什么主题应该有更多覆盖？
